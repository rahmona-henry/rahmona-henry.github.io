What 'scope' is, and how it works in javascript

Javascipt has two scopes: Global and Local. Global scope can be described as follows: Any variable declared outside of a function or a piece of code designed to do a specfic job is called a global variable and can be used by any piece of code in the program, there is no barriers or access restriction to that variable. On the other hand local scope is any variable that is declared inside a function. Local variablescan only be used by the functions they are declared within. No other piece of code apart from that function can access the local variable. Local variables are deleted when the function is completed. Global variables are deleted when you close the page.
